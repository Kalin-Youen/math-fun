# 小学数学乐园

一个专为小学生设计的数学学习辅助网站，覆盖小学1-6年级全部知识点。

## 功能特色

### 🎯 核心训练
- **速算练习** - 加减乘除限时挑战
- **九九乘法表** - 彩色乘法表 + 记忆小窍门
- **趣味数学** - 猜数字、找规律、比大小
- **应用题** - 故事化数学应用题练习

### 🧩 解题过程
- **竖式计算演示** - 一步一步看懂竖式
- **审题训练** - 标注关键词、判断运算方法
- **解题步骤** - 读题→分析→列式→计算→检验
- **易错题分析** - 典型错误案例 + 避坑指南

### 💡 概念理解
- **数感训练** - 数的组成、位值、估算
- **快速问答** - 限时抢答，训练反应速度
- **抽象概念** - 用图形理解抽象数学概念
- **数轴探索** - 在数轴上理解数的大小和运算

### 🔧 可视化工具
- **分数可视化** - 饼图、条形图理解分数
- **认钟表** - 互动时钟，学习看时间
- **公式卡片** - 周长、面积、体积公式速查
- **打印练习册** - 生成可打印练习题

### 📊 学习管理
- **学习仪表盘** - 进度追踪、薄弱环节、学习报告
- **每日一练** - 每天10道题+打卡，养成好习惯
- **错题本** - 自动收集错题，定期复习
- **口算闪卡** - 乘法口诀/公式闪卡快速记忆

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建后的静态文件在 `dist` 目录中。

## 部署

本项目已配置为静态导出，可直接部署到任何静态托管服务：
- Gitee Pages
- GitHub Pages
- Vercel
- Netlify
